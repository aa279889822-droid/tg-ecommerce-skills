# Visual Rules

## Main Image As Source Of Truth

The main image defines the listing's visual system. Before designing supporting images or the A+ page, lock:

- Product positioning.
- Primary buyer promise.
- Color palette.
- Lighting and realism level.
- Product angle and crop logic.
- Background cleanliness.
- Premium/value tone.
- Local market assumptions.

All later assets must inherit this direction unless the user chooses to change strategy.

## Five Main-Image Style Directions

Generate five genuinely different directions, selected from the product and competitor context. Examples:

- Clean compliant white-background hero.
- Premium studio realism.
- High-contrast conversion-focused ecommerce.
- Lifestyle-adjacent product hero, if platform rules allow.
- Technical/spec-focused product clarity.
- Natural material/detail close-up.
- Giftable/premium packaging focus.
- Compact mobile-first TikTok Shop clarity.

Do not force all five into the same palette or composition. Each direction should teach the user a real strategic choice.

## Main Image Rules

Default to marketplace-safe main-image logic:

- Clear product-first composition.
- High resolution and sharp details.
- Accurate product color, quantity, accessories, and packaging.
- No misleading props, unauthorized badges, fake UI, exaggerated claims, or competitor marks.
- White or clean background when required by platform/category.
- Product fills the frame enough for mobile browsing while preserving full visible shape.
- Shadow and perspective must look real, not pasted.

If TikTok Shop is the target, still use Amazon-grade clarity unless the user asks for a more social-commerce style.

## Supporting Image Rules

Each supporting image should answer one buyer objection. Avoid duplicate messages.

Common image roles:

- What it solves.
- Why it is better.
- How to use it.
- Where it fits in daily life.
- What is included.
- What size/spec/compatibility matters.
- Why it is safe/reliable.
- Why after-sales risk is low.

Use short localized copy. A strong supporting image usually needs one headline, one subline, and 2-4 labels, not a crowded poster.

## Localized Model Rules

Whenever a visual includes a person, hand, face, body, skin, hair, clothing, or lifestyle environment, match the model and scene to the target country. This applies to every category, not only beauty products.

- Use models who plausibly fit the target marketplace's local buyer base. For multicultural markets such as Malaysia and Singapore, choose natural local diversity or neutral close-up usage visuals instead of one generic foreign look.
- Match local household details, climate, bathroom/vanity style, clothing modesty, grooming style, and daily routines to the target country.
- Avoid stereotypes, costume-like styling, flags, landmarks, religious symbols, political symbols, or ethnic decoration unless the product itself makes them necessary and the user confirms it.
- For hands-only images, still localize skin tone, nail style, grooming, and usage context.
- If the target country is unknown, use neutral, globally acceptable models and state that localization is provisional.

## Scene Variety Rules

Do not let the image set become a sequence of product-on-background layouts. Build richer visual variation while keeping one consistent style system.

Use a mix of these scene types when relevant:

- Clean hero product image.
- Feature close-up or macro detail.
- Real use by a localized model.
- Multiple safe use positions or body areas when the product supports them.
- Home, bathroom, vanity, office, travel, outdoor, storage, gift, setup, and comparison scenes.
- Packaging or included-items scene.
- Trust or after-sales reassurance scene.

For personal care, beauty tools, massage tools, fitness tools, and wearable products, show different safe use zones only if they are supported by the product and do not create compliance risk. Example: a massage roller may show cheek, jawline, neck, shoulder, arm, or leg use only with safe language such as "daily care", "massage routine", "use area", or "home spa"; never pair those visuals with slimming, medical, treatment, circulation, body-shaping, or before/after claims unless the user provides compliant substantiation and the platform allows it.

For all categories, each supporting image should have a distinct job. If two proposed images communicate the same thing, replace one with a different scene or buyer objection.

## A+ / Detail Page Rules

Design for scroll logic:

- First screen: clarify value fast.
- Middle: prove features and use cases.
- Later: compare, reassure, and answer objections.
- End: after-sales reassurance and final buying confidence.

A+ modules should feel like one visual system. Repeat typography, product crop style, icon style, color accents, and lighting logic.

## Product And Competitor Analysis

Analyze competitor images before deciding the design:

- Repeated visual conventions in the category.
- Which product angles competitors use.
- Which benefits they over-explain or ignore.
- Color patterns and emotional tone.
- Trust devices and claims.
- Weaknesses: crowded text, unclear scale, fake scenes, bad localization, low-detail product rendering.

Turn competitor weaknesses into our image opportunities.

## Image Prompt Structure

Every image prompt should include:

- Product identity and exact visible details from source materials.
- Composition and camera angle.
- Background and environment.
- Lighting and realism level.
- Local-market visual setting if relevant.
- On-image text language and placement.
- Size/aspect ratio.
- Editing constraints: preserve product shape, color, logo, packaging, and accessory count.

Every prompt must also include a negative prompt using `compliance-and-defect-check.md`.
