# Output Blueprint

Use this structure for the full UGC director package.

## 1. Project Settings

Include:

- Product:
- Target country:
- Publishing platform:
- Duration:
- Language/accent:
- Selected UGC style:
- Main conversion goal:
- Available assets:
- Assumptions:

## 2. Product and Audience Analysis

Include:

- Product category
- Visible product details
- Likely buyer
- Buyer pain point
- Buyer desire
- Purchase trigger
- Main trust barrier
- Claims to avoid

Use "inferred" when details are guessed from the image.

## 3. UGC Style Recommendation

Give:

- Primary style
- Why it fits
- Backup style
- When to switch to the backup style

## 4. Virtual Creator Persona

Define:

- Age range
- Gender or neutral option
- Location/cultural context
- Personality
- Scene
- Outfit
- Camera feel
- Speaking tone
- Trust signal

Keep the creator realistic. Avoid over-polished model or commercial ad vibes unless the user requests them.

## 5. Hook Options

Generate 5 hooks:

- 2 direct pain-point hooks
- 1 curiosity hook
- 1 comparison hook
- 1 creator-style casual hook

Each hook should fit the target language and platform.

## 6. Voiceover Script

Write the main voiceover in the target language.

Also provide a Chinese explanation when the user is Chinese-speaking:

- Original target-language script
- Chinese meaning
- Delivery notes

Use short spoken lines. Avoid stiff translation.

## 7. Shot List

Use a timestamp table:

| Time | Visual | Action | Voiceover | On-screen text | Notes |
| --- | --- | --- | --- | --- | --- |

For 15 seconds, use 4-6 shots.
For 30 seconds, use 6-9 shots.
For 45 seconds, use 8-12 shots.

## 8. Jimeng Image Prompts

Provide separate prompt blocks:

1. Creator image prompt
2. Product-in-hand image prompt
3. Optional lifestyle scene prompt

Each block should include:

- Prompt
- Negative prompt
- Aspect ratio
- Notes for uploaded reference images

## 9. Jimeng Video Prompts

Provide one prompt per shot that needs generation.

Each prompt block should include:

- Reference images needed
- Subject tag: character
- Product tag: product
- Motion
- Camera
- Scene
- Duration target
- Anti-clipping note
- Negative prompt

## 10. Editing Script

Include:

- Clip order
- Cut rhythm
- Text overlay
- Sound/music suggestion
- B-roll placement
- CTA placement
- Export format

## 11. Subtitle Structure

Include:

- SRT-style timing
- Subtitle text in target language
- Optional Chinese translation
- On-screen text separate from spoken subtitles

## 12. Beginner Execution Checklist

End with a simple execution list:

1. Upload product image to Jimeng.
2. Generate creator image.
3. Generate product-in-hand image.
4. Generate video clips by shot.
5. Assemble clips in order.
6. Add subtitles and on-screen text.
7. Export vertical video.
8. Review with QC checklist.

## 13. Quality Control Checklist

Check:

- Product is visible and recognizable.
- Product label is not distorted if label matters.
- Hands and face are natural.
- Product, hands, clothing, hair, body, straps, collars, furniture, and props do not clip through each other.
- Wearable or pinned products sit on the visible surface with a plausible contact point.
- Hook appears in the first 1-3 seconds.
- Voiceover matches target country language.
- Claims are not exaggerated.
- CTA is clear.
- Video does not feel like a glossy brand ad unless requested.
